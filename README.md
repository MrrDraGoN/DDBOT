# Files

# Configuration

## CONFIG
| name| description|
|--|--|
| prefix | prefix komend |
| cleartime | usuwanie wiadomości bota pod x sekund |
|ModLogChannelID| kanał do logów bota |
| botstatus | status bota |
| botstatustype | typ statusu bota WATCHING | PLAYING | LISTENING |
